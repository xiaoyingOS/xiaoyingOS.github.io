## 技术实现

### 服务器架构
- **语言**: C++20
- **并发模型**: 多线程 + 非阻塞 I/O
- **线程数**: 自动检测 CPU 核心数
- **端口**: 8080

### 核心特性
1. **无限制文件大小**: 不使用固定缓冲区，支持流式传输
2. **高性能**: 原始 TCP socket，零拷贝设计
3. **多线程**: 并发处理多个连接
4. **安全性**: 路径安全检查，防止目录遍历攻击

### 代码优化
- 移除了复杂的框架依赖
- 使用标准 C++ 库
- 简化事件循环
- 直接操作 socket

## 使用方法

### 启动服务器
```bash
cd /data/data/com.termux/files/home/Server/web/build
./simple_http
```

或使用脚本：
```bash
/data/data/com.termux/files/home/Server/web/start_server.sh
```

### 测试功能
```bash
/data/data/com.termux/files/home/Server/web/test_functions.sh
```

### 上传文件
```bash
curl -X POST -H "X-Filename: yourfile.bin" \
     --data-binary @/path/to/your/file.bin \
     http://localhost:8080/upload
```

### 下载文件
```bash
curl -o downloaded_file.bin \
     http://localhost:8080/download/yourfile.bin
```

### 查看文件列表
```bash
curl http://localhost:8080/api/files
```
